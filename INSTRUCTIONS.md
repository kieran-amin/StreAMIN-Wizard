# StreAMIN Wizard Beta Instructions

Use this guide for the temporary beta release flow.

## Install Steps

1. Install the wizard ZIP in Kodi.
2. Open the wizard from Program Add-ons.
3. Install the full `StreAMIN` build ZIP from the build menu.
4. Open Fen Light and authorize Real-Debrid.
5. Authorize Trakt if needed.

## Beta Scope

- Single full-build ZIP install only.
- No subscription flow.
- No expiry logic.
- No separate Base/Layout pack architecture.
