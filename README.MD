# KieranWizard

Welcome to the KieranWizard repository.

### Installation
1. Download this repository as a ZIP file (Click the green '<> Code' button -> 'Download ZIP').
2. Open Kodi and navigate to **Add-ons** -> **Install from zip file**.
3. Select the ZIP file you downloaded.

Detailed setup instructions, including account authorization and troubleshooting, are included in the **INSTRUCTIONS.md** file inside the ZIP or can be viewed in the repository.
