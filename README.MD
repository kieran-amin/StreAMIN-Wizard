# KieranWizard

Welcome to the KieranWizard repository. This wizard installs and manages the KieranWizard Kodi build.

## Quick Installation

1. Click the green **`<> Code`** button above → **Download ZIP**.
2. **Do not unzip the file.** Open Kodi and go to **Add-ons** → click the **📦 open box icon** (top-left) → **Install from zip file**.
3. Navigate to your **Downloads** folder and select the downloaded zip file.
4. Wait for the **"KieranWizard Add-on installed"** notification.
5. Restart Kodi — the wizard will launch automatically and guide you through the rest.
